
# API REST com Spring e Swagger

## Descrição

Projeto de uma API REST desenvolvida com **Spring Boot**, documentada com **Swagger/OpenAPI** para facilitar a exploração e testes dos endpoints.

## Tecnologias

- **Java** - Linguagem de programação
- **Spring Boot** - Framework web
- **Swagger/OpenAPI** - Documentação automática de APIs
- **Maven** - Gerenciamento de dependências


## Estrutura do Projeto

```
src/
├── main/
│   ├── java/
│   │   └── com/seu/package/
│   │       ├── controller/
│   │       ├── service/
│   │       └── model/
│   └── resources/
│       └── application.properties
└── test/
```
